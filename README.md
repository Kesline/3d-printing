# 3D-Printing  
**Machine Learning-Driven Optimization of PLA 3D Printing Parameters**

This repository contains the research project conducted by **Kesline Jean Baptiste** and **Jorwin Reyes** as part of the 2024 **IDS Student Fellowship Program**, funded by the **Sloan Foundation**. The project was presented at the **Massachusetts Undergraduate Research Conference (MassURC)** at UMass Amherst.
